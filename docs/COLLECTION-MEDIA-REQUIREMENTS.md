# Collection Media Requirements

Temporary Day 3 assets reuse homepage SVG placeholders under `apps/web/public/media/home/collections/`.

## Shared guidelines

| Item | Recommendation |
|------|----------------|
| Hero desktop | 1920×1080 or 1600×900, JPG/WebP, ≤400KB ideal / ≤1MB max |
| Hero mobile | 1080×1350 or 750×1200, JPG/WebP, ≤250KB |
| Story stills | 1600×1200, JPG/WebP, ≤350KB |
| Optional video | H.264, `+faststart`, muted loop, ≤5MB |
| Aspect (hero) | ~16:9 desktop, ~4:5 mobile |
| Focal point | Keep subject in safe center; leave lower third clear for title |
| Safe text area | Bottom 35% of hero for title/subtitle |
| Rights | Original IPLAY or licensed only — never ZURU assets |
